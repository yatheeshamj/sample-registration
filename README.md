# spotify Registration

spotify registration to class sign up flow for web

## Tech/Framework Used

__Built With__

* [Create React App](https://github.com/facebook/create-react-app)
* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [Redux Saga](https://github.com/redux-saga/redux-saga)

## Development Setup

__Installing dependencies:__

* Install node.js ^10.11.0: [https://nodejs.org/](https://nodejs.org/)
* Install npm ^6.4.1
* Install packages: `npm i`
  
__Running the application in development mode:__  
  
* `npm start`, then go to: [http://localhost:3000](http://localhost:3000)

## Devtools

spotify Registration uses Redux for state management; baked in access to Redux Devtools. Use the browser extension version and not the self implemented version of Redux Devtools. [Go Here](https://github.com/zalmoxisus/redux-devtools-extension) to learn more and installation instructions. Just install the extension, activate it, and you're all set.

## Building Static Assets

__Create an optimized production build:__

Static assets will be created in the build folder

* `npm run build`
